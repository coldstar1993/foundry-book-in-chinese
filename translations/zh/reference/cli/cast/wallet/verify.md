# cast wallet verify

验证信息的签名

```bash
$ cast wallet verify --help
用法： cast wallet verify --address <ADDRESS> <MESSAGE> <SIGNATURE>

参数：
  <MESSAGE>    The original message
  <SIGNATURE>  The signature to verify

选项：
  -a, --address <ADDRESS>  The address of the message signer
  -h, --help               Print help
```
