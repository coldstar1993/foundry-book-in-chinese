# cast to-rlp

RLP 编码十六进制数据，或一个十六进制数据的数组。

```bash
$ cast to-rlp --help
```

```txt
用法：cast to-rlp [VALUE]

参数：
  [VALUE]
          The value to convert.
          
          This is a hex-encoded string, or an array of hex-encoded strings. Can
          be arbitrarily recursive.

选项：
  -h, --help
          Print help (see a summary with '-h')
```
