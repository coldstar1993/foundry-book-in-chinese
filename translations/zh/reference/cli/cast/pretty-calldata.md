# cast pretty-calldata

美化打印 calldata。

```bash
$ cast pretty-calldata --help
```

```txt
用法：cast pretty-calldata [OPTIONS] [CALLDATA]

参数：
  [CALLDATA]
          The calldata

选项：
  -o, --offline
          Skip the https://openchain.xyz lookup

  -h, --help
          Print help (see a summary with '-h')
```
