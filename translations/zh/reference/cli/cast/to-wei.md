# cast to-wei

将 ETH 转换为 wei。

```bash
$ cast to-wei --help
用法： cast to-wei [VALUE] [UNIT]

参数：
  [VALUE]
          The value to convert

  [UNIT]
          The unit to convert from (ether, gwei, wei)
          
          [default: eth]

选项：
  -h, --help
          Print help (see a summary with '-h')
```
