# cast 4byte-decode

使用 https://openchain.xyz 解码 ABI 编码的 calldata

```bash
$ cast 4byte-decode --help
```

```txt
用法：cast 4byte-decode [OPTIONS] [CALLDATA]

参数：
  [CALLDATA]  The ABI-encoded calldata

选项：
  -h, --help  Print help

Display options:
  -j, --json  Print the decoded calldata as JSON
```
