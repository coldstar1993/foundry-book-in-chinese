# cast send --create

用于部署原始合约字节码

```bash
$ cast send --create --help
```

```txt
用法： cast send --create <CODE> [SIG] [ARGS]...

参数：
  <CODE>     The bytecode of the contract to deploy
  [SIG]      The signature of the function to call
  [ARGS]...  The arguments of the function to call

选项：
  -h, --help  Print help
```
