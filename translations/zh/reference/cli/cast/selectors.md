# cast selectors

从字节码中提取函数选择器和参数

```bash
$ cast selectors --help
```

```txt
Usage: cast selectors [OPTIONS] <BYTECODE>

Arguments:
  <BYTECODE>  The hex encoded bytecode

Options:
  -r, --resolve  Resolve the function signatures for the extracted selectors
                 using https://openchain.xyz
  -h, --help     Print help
```
