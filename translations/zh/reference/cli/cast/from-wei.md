# cast from-wei

将 wei 转换成 ETH 数量。

```bash
$ cast from-wei --help
用法： cast from-wei [VALUE] [UNIT]

参数：
  [VALUE]
          The value to convert

  [UNIT]
          The unit to convert from (ether, gwei, wei)
          
          [default: eth]

选项：
  -h, --help
          Print help (see a summary with '-h')
```
