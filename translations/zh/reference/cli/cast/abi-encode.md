# cast abi-encode

对给定的函数参数进行 ABI 编码，不包括选择器

```bash
$ cast abi-encode --help
```

```txt
用法：cast abi-encode [OPTIONS] <SIG> [ARGS]...

参数：
  <SIG>      The function signature
  [ARGS]...  The arguments of the function

选项：
      --packed  Whether to use packed encoding
  -h, --help    Print help
```
