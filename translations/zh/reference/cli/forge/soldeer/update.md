# forge soldeer update

通过读取配置文件更新依赖项

```bash
$ forge soldeer update --help
```

```txt
用法: forge soldeer update [选项]

选项:
  -g, --regenerate-remappings  如果设置，此命令将删除现有的重映射并重新创建它们
  -d, --recursive-deps         如果设置，此命令将递归安装依赖项（通过子模块或通过
                               soldeer）
  -h, --help                   打印帮助信息

欲了解更多信息，请阅读 README.md
```
